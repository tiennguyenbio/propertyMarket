# Suburb Profile

## How to install the package

### Install from the repo
`git clone https://github.com/tiennguyenbio/propertyMarket.git
cd suburb_profile
uv pip install -e . 
`
### Install from github
`uv pip install git+https://github.com/tiennguyenbio/propertyMarket.git#subdirectory=suburb_profile
`
